# NEXFM Website

Coming-soon site for **NEXFM** — an operating partner connecting people, places, and performance.

## Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS v4
- pnpm

## Getting started

```bash
pnpm install
pnpm dev
```

Open [http://127.0.0.1:3000](http://127.0.0.1:3000).

## Scripts

- `pnpm dev` — local development
- `pnpm build` — production build
- `pnpm start` — serve production build
- `pnpm lint` — ESLint
